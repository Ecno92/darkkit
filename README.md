# darkkit

Personal kit of Lua scripts for [darktable](https://www.darktable.org/).

> Scripts in this repository may contain AI-assisted code.

## Scripts

- **print_image** — Print images directly via CUPS. Lets you pick printer, paper size, and scaling from within darktable's export panel. Optimised for the Canon Selphy CP1500.

## Prerequisites

Make sure to first install the community scripts and the script manager:[darktable community Lua scripts](https://github.com/darktable-org/lua-scripts).

## Installation via Script Manager

darktable's built-in Script Manager can install scripts from third-parties.

1. Open darktable and switch to the **lighttable** view.
2. In the lower-left panel, find the **Script Manager** and expand **"install/update scripts"**.
3. Enter the following values:
   - **URL**: `https://github.com/Ecno92/darkkit.git`
   - **Folder**: `darkkit`
4. Click **"install additional scripts"** and restart darktable.
5. In Script Manager, start/stop scripts, select the **darkkit** folder and enable the scripts you want.
